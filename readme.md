
## Requirements

### Python 3.13.2 
[link](https://www.python.org/downloads/release/python-3132/)

### Virtual enviroment
create
```bash  
python -m venv venv
``` 
activate
```bash  
.\venv\Scripts\activate
```

### Libraries
```bash
pip install -r requirements.txt
```
### Start server
```bash
python server.py
```
### Run client
```bash
python client.py
```
